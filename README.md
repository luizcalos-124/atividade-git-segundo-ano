# Atividade de Programação em Python
Uma escola está organizando um evento especial para estudantes. Para participar, o aluno precisa atender a algumas regras.

## Regras para entrada no evento:

### O aluno pode entrar se:
* Tiver idade maior ou igual a 16 anos E tiver autorização dos responsáveis,
OU
* Tiver 18 anos ou mais.

## O que o programa deve fazer:

### Pedir ao usuário:
* idade do aluno
* se possui autorização dos responsáveis (sim ou não)
* Mostrar uma mensagem informando se a entrada foi permitida ou negada.
* "Entrada permitida apenas com responsável" quando o aluno tiver menos de 16 anos, mas estiver acompanhado de um responsável.

## Exemplo de execução
Digite sua idade: 17
Possui autorização dos responsáveis? (sim/nao): sim
Entrada permitida no evento.

Digite sua idade: 15
Possui autorização dos responsáveis? (sim/nao): nao
Entrada não permitida no evento.

Digite sua idade: 15
Possui autorização dos responsáveis? (sim/nao): sim
Entrada permitida apenas com responsável.
